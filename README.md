# FIFA 26 Tournaments Management System

نظام شامل لإدارة بطولات FIFA 26 مع ثلاثة أنواع مختلفة من البطولات وواجهة إدارة متقدمة.

## الميزات الرئيسية

### الصفحات الرئيسية
- **الصفحة الرئيسية**: عرض جميع البطولات مع بنر إعلانات متحرك ونموذج تسجيل سريع
- **صفحة الدوري**: نظام دوري كامل مع جدول الترتيب والمباريات
- **صفحة الأونلاين**: بطولة أونلاين (FUT) مع نفس نظام الترتيب
- **صفحة الخروج المغلوب**: بطولة حضورية مع قوس البطولة

### لوحة التحكم للإدمن
1. **إدارة المشتركين**: الموافقة على المشتركين أو رفضهم
2. **إدارة النتائج**: تحديث نتائج المباريات وحساب الترتيب تلقائياً
3. **إدارة البطولات**: إنشاء وتعديل وحذف البطولات
4. **إدارة الإعلانات**: إضافة وتعديل الإعلانات المتحركة
5. **الإعدادات**: تحديث معلومات التواصل والإعدادات العامة

## التكنولوجيا المستخدمة

- **Frontend**: HTML5, CSS3, JavaScript (ES Modules)
- **Database**: Supabase (PostgreSQL)
- **Build Tool**: Vite
- **Design**: Material Design principles

## هيكل قاعدة البيانات

### الجداول الرئيسية
- **tournaments**: معلومات البطولات
- **participants**: المشتركين في البطولات
- **matches**: المباريات والنتائج
- **standings**: جدول ترتيب الدوري
- **bracket**: قوس البطولة (للخروج المغلوب)
- **advertisements**: الإعلانات
- **settings**: إعدادات الموقع

## نظام حساب النتائج

### نظام الدوري
- الفوز = 3 نقاط
- التعادل = 1 نقطة
- الخسارة = 0 نقطة
- يتم الترتيب حسب:
  1. النقاط الكلية
  2. الفرق بين الأهداف المسجلة والمستقبلة
  3. عدد الأهداف المسجلة

### نظام الخروج المغلوب
- يتم إنشاء قوس البطولة تلقائياً
- يتم ترتيب المشتركين عشوائياً
- يتم تحديد الفائز بناءً على نتيجة المباراة

## الأمان

- تم تفعيل Row Level Security (RLS) على جميع الجداول
- السياسات الأمنية تحمي بيانات المستخدمين
- الوصول العام محدود للبيانات غير السرية

## البدء

```bash
# تثبيت المتطلبات
npm install

# تشغيل الخادم التطويري
npm run dev

# بناء للإنتاج
npm run build
```

## التصفح

- الصفحة الرئيسية: `index.html`
- صفحة الدوري: `league.html`
- صفحة الأونلاين: `online.html`
- صفحة الخروج المغلوب: `knockout.html`
- لوحة التحكم: `admin.html`

## الملفات المهمة

- `config.js`: إعدادات الاتصال بـ Supabase
- `styles.css`: جميع الأنماط
- `js/main.js`: منطق الصفحة الرئيسية
- `js/admin.js`: منطق لوحة التحكم
- `js/utils.js`: دوال مساعدة (حساب الترتيب، إنشاء القوس)
